Swift 中 Substring 的方法使用

发表于 2018-03-23 | 分类于 iOS/macOS ， Swift |
字数统计: 939 | 阅读时长 ≈ 0:01

在我初学 Java/C++ 时，经常遇到处理字符串的问题，很多都是截取字符串或者对字符串进行特定的处理。超级多的语法糖写起来很是方便，我曾经也是相信 Swift 在 String 类型上有很多的干货。但在 Swift 4 之前，其对字符串截取依然不是很方便。

最近写一个实验的时候发现了 Swift 4 里出现了一个叫 Substring 的新特性，了解 Java 的我就一下明白这个新特性的作用。于是便看了一下文档。在 Swift 4 里 String 是一个集合类型了，可以同 Array 等 Collection 一样方便的操作方式。

Substring 中的几个实用方法

dropFirst
